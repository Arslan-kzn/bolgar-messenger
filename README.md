# BOLGAR Messenger

BOLGAR Messenger — Matrix-клиент, ориентированный на татарскую и мусульманскую общину.  
Проект основан на FluffyChat и распространяется по лицензии **AGPL-3.0**.

## Статус проекта
- Активная разработка: ветка `bolgar-branding`
- Публичный репозиторий: исходный код открыт

## Основная идея
Приложение является клиентом Matrix: оно подключается к Matrix-серверу и отображает данные, которые сервер возвращает.

Сервер по умолчанию (для нашего сообщества): **bolgarlar.ru**
